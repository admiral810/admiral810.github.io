Project Work

Housing Market Insights

#Python #JavaScript #HTML #CSS #SQL

What It Is: This is an interactive web deplyed app that you can go to linked HERE. It allows a user to browse various housing market insights by US county such as median rent, median home value, median income, and other measures.

How It's Built: This project leveraged a number of tools, using a python flask app to serve data to the webpage from a SQL database. More detail on the technical aspects can be found in its Github ReadMe.


Vizualization: This displays the ability to "play" a bubble chart over time, showing how rent prices outpaced income in many counties in California. More vizualizations can be found in the app.



IPO Stocks Database

#Python #SQL #SQLAlchemy

What It Is: This is a project that pulls IPO stock information using Python scripts from the web and houses it in a SQL database. Information includes dates when a company is going public, performance results, and company characteristics.

How It's Built: This project leverages Python scripts to pull information from the web-including Nasdaq and Yahoo Finance-to add and update the database. Stored procedures in SQL are also leveraged to add value to the dataset. More information can be found in its Github ReadMe.



Vizualization: This shows an ER diagram for the current schema setup of the SQL database. In total this database houses over 300,000 records of IPO stock information.



Predictive Laptop Prices

#Python #MachineLearning #JavaScript #HTML #CSS

What It Is: This is an interactive web deplyed app that you can go to linked HERE. It is a gameification deployment where a user guesses a laptop price based on a number of attributes against 3 machine trained models.

How It's Built: This project leveraged linear regression, random forest, and neural network models using Python. More detail on the technical aspects can be found in its Github ReadMe.



Vizualization: The user may reference a laptop's attributes to inform their guess of a laptops price, and 3 machine learning trained models do the same. Play for yourself HERE.



Category Management Dynamic Report

#AdvancedExcel #DAX #SQL

What It Is: A dynamic report that allows users to browse food category information in an interactive format. Users can get customized perspectives on category performance, competitors, pricing, attributes, and other category insights.
